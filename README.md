## 安装

1. 下载并安装 [Tampermonkey](https://www.tampermonkey.net/) 或其他的用户脚本管理器。
2. 下载仓库中的 `XMWmax.user.js` 文件。
3. 打开 Tampermonkey 并点击「添加新脚本」。
4. 选择下载的 `XMWmax.user.js` 文件并点击「安装」。

## 设置

1. 打开小码王网页。
2. 点击右下角的“XMWmax”按钮。
3. 点击“添加插件”。
4. 手动导入插件或通过URL导入插件。
